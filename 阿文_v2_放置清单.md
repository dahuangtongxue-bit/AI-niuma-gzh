## 环境变量（与阿桃同一套逻辑）
- LLM_API_BASE / LLM_API_KEY / LLM_MODEL：deepseek 官方那套（写文主力）
- LLM_MODEL_JUDGE：留空
- VISION_API_BASE = https://api.minimaxi.com/v1
- VISION_API_KEY = MiniMax 官方 key
- LLM_MODEL_VISION = MiniMax-M3（走截图入职必配）
- 头图 AI 兜底如需生图：IMAGE_MODEL / IMAGE_SIZE（投喂用真实照片时不需要）

## 这次补了什么（对齐阿桃）
1. **加固内核**：/api/chat 流式直通（治 504）；pipeline 防弹 JSON 解析、思考模型免疫（stripThink）、成文与评分串行（避 429）、失败重试、停止按钮。
2. **真实档案**：REALITY_RULE 铁律——禁编造名称/地址/业务，本主体视角，文末挂载引流信息。入职支持三入口（链接/截图/手填），读图复用 MiniMax-M3。
3. **投喂模式**：工作台「📸 投喂一篇 / 🗓 自主头条」切换。投喂模式跳过选题海选，按你给的主题+文风+结构+照片精做一篇；头图优先用你的真实照片（C_PHOTO 模板），无图才 AI 兜底。
4. **发布备料**：顶部一键复制带排版正文（富文本，贴后台即有格式）+ 标题 + 摘要；头图+金句卡 zip 打包下载（阿文本就有，比阿桃更强）。

## 验收（一把抓牛肉面）
1. 入职：传几张点评页/菜单/门头截图 → 阿文读出档案 → 核对修改 → 入职（选主题色）
2. 投喂模式：传 1-2 张真实照片 + 主题「我们为什么坚持每天凌晨现熬汤」+ 文风「温暖治愈」→ 写这一篇
3. 看产出：长文紧扣主题、真实信息、文末带到店引导；头图是你的真实照片叠标题；右侧质检逐条机检
4. 发布：复制带排版正文→公众号后台粘贴（带格式）；打包下载头图金句卡→上传；群发
5. 切「自主头条」点开工，对比自主选题模式
